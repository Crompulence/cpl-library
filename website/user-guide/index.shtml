Step-by-step guide: coupling with CPL Library

The aim of the following text is to lead the user right from the start to the end: from downloading CPL Library, all the way through to running and analysing a coupled simulation. Please get in touch if something is unclear or just plain wrong!

Pre-requisites

Before you begin, make sure you have the following installed on your system:
  • A C/C++ compiler that supports the C++14 standard,
  • A Fortran compiler that supports the F2008 standard, and
  • An MPI library
CPL Library was developed and tested using the GCC compiler collection and MPICH, which are both free and open-source.

Download and unzip

Assuming you're compiling on linux, the latest zip file can be obtained from github using wget:

wget https://github.com/Crompulence/cpl-library/archive/master.zip

Alternativly, you can download with git or by clicking on the direct download link. Please see the download page. Next, we want to unzip the file into a folder called cpl-library

unzip ./master.zip

If you have problems with this, you may need to download unzip. For ubuntu, this can be obtained with sudo apt-get install unzip or equivalent for you distribution. Once you've unzipped the file, change directory to the cpl build directory ready to compile.

cd cpl-library-master/cpl/

Compilation

Compiling CPL-Library itself, as a statically-linked library, is likely to work simply by running Gnu make:

make all

This build has currently been tested using gfortran, gcc and mpich. Support for the intel compiler suite is under development. For more complicated builds, the template file should be changed as outlined in the README within the directory.

make PLATFORM=template cpl_

where "template" refers to the name of the included file /make/template.inc, in which the compilers and flags are defined. The template provided is likely to work for most machines, but the user is advised to create their own version.
