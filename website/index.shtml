Welcome to the CPL Library Website

This is the CPL library website. CPL library is software for linking together Computational Fluid Dynamics (CFD) and Molecular Dynamics (MD) software. CPL library is for a concurrent simulation where both molecular and continuum regions exist at the same time and length scale with information exchanged along a boundary. This type of coupled simulation was pioneered by O'Connell and Thompson i, with the first flux coupling simulation was proposed by Flekkoy et al , popularised by Nie et al and a useful review provided in Mohamed and Mohamad . Information is exchanged in an overlap region between processors on the CFD and MD sides. CPL library handles the mapping between the domains of the two codes and facilitates large scale simulations. This aims to allow unprecidented coupled simulation scales. CPL library also aims to lower the boundary of entry for new researchers in coupled simulation and to make this technique avilable to industry. Open source sofware, for example LAMMPS and OpenFOAM, are currently under development to be coupled using CPL library allowing users of either code to extend thier own research and facilitates collabortation between a range of research areas.
