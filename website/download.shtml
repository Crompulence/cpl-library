Downloading the latest releases of CPL Library

Version 1.0.0 of CPL library is now avialable to download from . This includes a remodelling of the interface from version 0.9.0 so will break backward compatibility. The interface has been greatly simplified so moving to the new version should hopefully be fairly straightforward.

Bleeding edge CPL Library

The latest version of CPL library may also be downloaded from github git. Copy and paste the following command into your Terminal:

git clone https://github.com/Crompulence/cpl-library.git

Or see our github homepage. Alternatively, a zip file may be downloaded from here. There is a master and dev branch, the master branch is tested using Travis CI and should be free from basic bugs while the dev will often be unstable and used for testing new features.
